Configuración de la base de datos:

-Ejecutar el archivo BDD en SQL Server y asegurarse de ejecutar cada uno de los scripts incluidos.

Configuración del entorno de desarrollo:

-Abrir Visual Studio.
-Cambiar el servidor de base de datos SQL al que está instalado en nuestro computador en los siguientes archivos:

-GestionProductosContext.cs (Línea 27)
-ProductController.cs (Línea 12)
-appsettings.json (Línea 10)
-Guardar los cambios realizados.

Ejecución de la solución:

-Guardar y acceder a la base de datos configurada.
-Ejecutar la solución en Visual Studio y ejecutar la API.

Configuración del entorno de prueba:

-Abrir Visual Studio Code.
-Abrir el folder del proyecto en Visual Studio Code.
-Dentro de la carpeta "Front/HTML", abrir el archivo HTML correspondiente.

Siguiendo estos pasos, el proyecto estará configurado correctamente y podrás trabajar en él sin problemas.
